## Setup

Prerequisites:

- Node.js 22.x (see `.nvmrc`)

Enable Corepack and install dependencies:

```bash
corepack enable
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Formatting

```bash
pnpm format
pnpm format:check
```
