A specialized tool for teachers that transforms school grade spreadsheets into professional presentation materials for parent-teacher conferences.

## Features

- XLSX import from Polish school systems
- HTML presentation generation
- Customizable templates
- Privacy-focused local processing

## Use Case

Teachers in Polish schools often need to present student progress during parent-teacher meetings. This tool automates the creation of clear, professional presentations from the standard grade export format used by Polish school management systems.
